9 полнофункциональных модулей — от дашборда с аналитикой до чата и базы знаний.

📊
Дашборд и аналитика
Сводная статистика по выручке, клиентам, задачам. Воронка конверсии, сезонность, KPI сотрудников. Фильтр по периоду, экспорт Excel.
Выручка Конверсия KPI Excel
👥
Воронка и клиенты
CRM-база с полной историей. Статусы лид/активный/VIP. Источники трафика, фильтры, поиск, карточки клиентов. Визуальная воронка продаж.
Лиды VIP Воронка История
✅
Задачи / Kanban
Kanban-доска с 4 колонками: Новые → В работе → Проверка → Готово. Дедлайны, приоритеты, назначение на сотрудника. Бейдж просроченных.
Kanban Дедлайны Приоритеты
🔧
Заявки на сервис
Учёт заявок на ремонт и обслуживание. Типы поломок, статусы, время в работе, суммы (TMT/USD). Гарантийный талон. Расписание.
Заявки Гарантия Расписание
🏗️
Монтаж
Заявки на монтаж с адресом, объёмом работ, статусами. Автоперенос завершённых в клиентский сервис для обзвона. KPI монтажников.
Монтаж Автоперенос KPI
💬
Чат
Корпоративный чат в реальном времени (WebSocket). Общий канал и личные переписки. Синхронизация между всеми подключёнными пользователями.
Realtime ЛС WebSocket
📚
База знаний
Корпоративная wiki с категориями и поиском. Статьи в rich-text формате. Доступна всем сотрудникам, редактирование администратором.
Wiki Поиск Категории
🧮
Калькулятор
Встроенный калькулятор для быстрых расчётов стоимости услуг, материалов, сделок. Сохранение шаблонов расчётов.
Расчёты Шаблоны
👤
Сотрудники и структура
Оргструктура компании, отделы, должности. Управление ролями (admin/user). Активация/деактивация, цветовые аватары, контакты.
Роли Структура RBAC

Надёжный стек

🔒
Безопасность
bcrypt (12 раундов), брутфорс-защита (5 попыток → 15 мин), сессии 24ч в SQLite, CSP, XSS-санитизация всех полей.
🗄️
SQLite / sql.js WASM
Работает без нативных компиляторов на Windows и Linux. WAL-журнал, пагинация LIMIT/OFFSET, автобэкапы.
⚡
WebSocket Realtime
Мгновенная синхронизация данных между всеми клиентами. Throttle 80мс, heartbeat, до 60 одновременных соединений.
📊
Excel экспорт
xlsx-js-style с поддержкой стилей ячеек. Экспорт клиентов, монтажа, сервиса, дашборда в .xlsx прямо из браузера.
📝
Winston + Ротация логов
Audit-log каждого действия. Ротация: новый файл каждый день, хранение 30 дней, архивирование, макс 50 МБ/файл.
🔄
Бэкапы FTP + SMB
Автоматические бэкапы каждые N минут с retry (3 попытки). Копирование на FTP и SMB-шары. Хранение 7 дней истории.
👑
RBAC — Роли и права
Серверная проверка ролей (admin/user). Только admin видит структуру и сотрудников. Bulk-set защищён от рядовых пользователей.
🌐
REST API v1 + Webhooks
Полноценное REST API с X-API-Key аутентификацией. Rate-limit 100 req/15мин. Webhooks для звонков и email. PM2-ready.
